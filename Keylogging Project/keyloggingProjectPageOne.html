Overview

The practice of using a software program or hardware device (keylogger) to record all keystrokes on a computer keyboard, either overtly as a surveillance tool or covertly as spyware: Many employers are making use of keylogging to monitor their employees' computer habits. -Dictionary.com

While the programs themselves are legal, with many designed to allow employers to oversee the use of their computers, keyloggers are most often used for stealing passwords and other confidential information.

Why so Dangerous?

Most keyloggers are not stopped by HTTPS encryption because that only protects data in transit between computers; software-based keyloggers run on the affected user's computer, reading keyboard inputs directly as the user types.

Keylogger program authors may be able to update their program's code to adapt to countermeasures that have proven effective against it.

The pieces of code Keyloggers use are very short and light, because they don't do much. So it is hard to detect.
